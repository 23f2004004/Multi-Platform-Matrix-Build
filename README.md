# GitHub Actions Matrix Build Challenge

This repository demonstrates a modern CI/CD pipeline utilizing **GitHub Actions matrix strategy** for parallel builds and robust **artifact management**.

## Challenge Compliance Details

* **Matrix Strategy:** Configured to run builds across three different Node.js versions (18, 20, 21).
* **Parallel Execution:** Each variant runs as a separate, concurrent job.
* **Artifacts:** Each job generates a unique artifact named with the required prefix `build-132029c-`.
* **Step Identifier:** The required identifier `matrix-132029c` is included in a workflow step.
